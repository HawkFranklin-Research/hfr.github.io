2. Methodology: The Event-Driven Agentic Pipeline

1. User Instruction

High-level task in natural language via CLI (e.g., `preprocess`, `custom instruction`)

2. LLM-Powered Planning

gpt-5.1 & LlamaIndex plan sub-tasks and generate Python code dynamically.

3. Secure Execution

Code Interpreter tool runs code in a sandbox. Agent handles errors & logs events.

4. Validated Artifacts

Outputs models, plots (DCA, SHAP), and reproducible run manifests.
