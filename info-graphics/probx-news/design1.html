From Input to Verdict

ProbX doesn't just guess. It performs a multi-step forensic investigation using agentic workflows and a "mental spreadsheet" approach.

1. Ingestion

User submits a suspicious URL, text claim, or image. The agent parses metadata and visual features.

2. Agentic Search

Gemini searches the web, news archives, and X/Twitter feeds for corroboration, timestamps, and image history.

3. Fact Compilation

Findings are logged into a structured "Mental Excel Sheet" comparing claims vs. gathered evidence in real-time.

4. Verdict

A final confidence score is generated (Real/Fake/Satire) along with a source list and detailed markdown report.
