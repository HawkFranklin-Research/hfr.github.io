HFR Visual Template (v3)

Reusable building blocks to reproduce the v3 look on any page and retheme quickly via CSS variables.

What It Covers

  • Gradient highlight on selected words
  • Left‑tilted hue background behind hero
  • Announcement strip (thin gold bar)
  • Nav underline + sizes (matching v2/v3)
  • Product/research card hover and glow
  • Easy theming via CSS variables (one place to change colors)

How To Use It

  • Gradient word: wrap only the part to highlight with <span class="gradient-text">...</span>.
  • Left hue: add hero-left-hue to the hero section container (not the whole page).
  • Announcement: insert a thin section under the hero with announce-strip.
  • Nav: add nav-link to top menu anchors. Use text-base lg:text-lg for size.
  • Cards: add product-card or research-card to relevant containers.
  • Logo: apply logo-header in header and logo-footer in footer.

Re‑Theming (choose different colors)

  • Quick: override --accent-* values in :root.
  • Multi‑themes: keep gold in :root and add <body class="theme-emerald"> or theme-sky.
  • Palette examples: Emerald (#10b981), Sky (#0ea5e9).
  • Gradient word: tweak --grad-a (edge) and --grad-b (main).
  • Hue: adjust radial alphas ~0.06–0.18 and use your accent’s RGB for tint.
