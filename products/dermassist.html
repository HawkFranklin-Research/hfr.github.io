How DermAssist Works

Image Upload

Clinicians upload skin lesion images through an intuitive interface

AI Processing

CNN/Transformer pipelines analyze the image using dermatology datasets

Probability Assessment

System calculates likelihood scores for various conditions

Clinical Decision Support

Presents findings to assist (not replace) clinician diagnosis

Key Features

  • Multi-label predictions for complex dermatological conditions
  • Pluggable classifier backend (LogReg, MLP, etc.)
  • Top-K probability visualization for differential diagnosis
  • API integration for seamless workflow incorporation
  • Sample image library for testing and demonstration

Technical Specifications

  • Model Architecture: Hybrid CNN/Transformer pipelines
  • Training Data: ISIC, dermoscopy images, and proprietary datasets
  • Framework: PyTorch/TensorFlow with scikit-learn classifiers
  • Interface: Gradio UI with FastAPI backend
  • Compliance: Research/demo only (not a diagnostic device)

Supported Conditions

Eczema

Chronic condition causing inflamed, itchy, cracked, and rough skin

Psoriasis

Autoimmune disease that causes raised, red, scaly patches

Tinea

Fungal skin infection causing ring-shaped rashes

Contact Dermatitis

Skin irritation from contact with allergens or irritants

Herpes Zoster

Viral infection causing painful skin rash with blisters
